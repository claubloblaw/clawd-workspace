# OpenClaw Free Access Paths

## Core Truth
OpenClaw is free/open-source framework; spend comes from model provider access.

## Practical Paths
- **Gemini API free tier**: strongest free option for regular usage.
- **OpenRouter free models**: usable for low-cost/zero-cost experimentation.
- **OpenAI API**: generally paid/prepaid in modern setup (no reliable standing free-credit assumption).

## Why This Matters
Useful for onboarding people who want to test OpenClaw cheaply before committing to paid API usage.
